## Installation

```bash
$ npm install
```

## 1. RESTful API

```bash

```

| Method | Route            |
| ------ | ---------------- |
| POST   | /sign-up         |

| GET    | /me              |
| PUT    | /me              |
| DELETE | /me              |
| PUT    | /change-password |
