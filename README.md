## Installation

```bash
$ npm install
```

## 1. RESTful API

```bash
$ npm run start
```

Test API at http://localhost:3000 on route below

| Method | Route            |
| ------ | ---------------- |
| POST   | /sign-up         |
| POST   | /sign-in         |
| GET    | /me              |
| PUT    | /me              |
| DELETE | /me              |
| PUT    | /change-password |
